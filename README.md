# Retail Sales & Customer Insights Dashboard

## 📊 Project Overview

This project analyzes retail sales data to understand customer behavior, identify high-value customers, and uncover revenue trends using Python, SQL, and Power BI.

## 🛠 Tools Used

* Python (Pandas) – Data Cleaning
* SQL (MySQL) – Data Analysis
* Power BI – Dashboard

## 📈 Key Insights

* UK contributes the majority of revenue
* High-value customers generate the most revenue
* Many customers are at risk
* Sales vary across months
