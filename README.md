# 📊 Retail Sales & Customer Insights Dashboard

🚀 End-to-end data analytics project using Python, SQL, and Power BI to analyze retail sales data, identify customer behavior patterns, and generate actionable business insights.

---

## 📌 Project Overview

This project focuses on analyzing retail transaction data to uncover key business insights such as revenue trends, customer segmentation, and top-performing products.

The goal is to simulate a real-world business scenario where data is cleaned, analyzed, and visualized to support decision-making.

---

## 🛠️ Tools & Technologies

* **Python (Pandas)** → Data Cleaning & Preprocessing
* **SQL (MySQL)** → Data Analysis
* **Power BI** → Interactive Dashboard & Visualization

---

## 🔄 Project Workflow

1. Data Cleaning using Python
2. Data Analysis using SQL
3. Customer Segmentation using RFM Model
4. Dashboard Development in Power BI
5. Insight Generation & Business Recommendations

---

## 📂 Data Description

* `raw_data.csv` → Original dataset with missing values and inconsistencies
* `cleaned_retail_data.csv` → Processed dataset after cleaning and feature engineering

---

## 📈 Key Insights

* United Kingdom contributes the majority of total revenue
* High-value customers generate the highest revenue
* A significant portion of customers fall into the “At Risk” segment
* Sales show noticeable fluctuations across months
* A small group of customers contributes disproportionately to total revenue

---

## 💡 Business Recommendations

* Focus on retaining high-value customers through loyalty programs
* Target at-risk customers with personalized marketing strategies
* Expand strategies in high-performing regions (e.g., UK)
* Optimize product offerings based on top-selling items
